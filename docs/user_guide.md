# User Guide

Welcome to the EcoFlow Supply Chain Decarbonization platform.

## Key Dashboard Views

1. **Main Dashboard:** Highlights total Scope 3 carbon footprint, top emitting suppliers, and potential CBAM tariffs.
2. **Carbon Intelligence View:** Displays geographical emission intensity across global shipping pipelines.
3. **Supplier Network Graph:** Visualizes tier relationships between suppliers, highlighting critical targets for mitigation.
4. **CBAM Audit Panel:** Lists details of imported product groups that fall under EU carbon taxation rules.

## Interacting with the AI Assistant

Use the chatbot panel on the dashboard to execute commands using natural language. Try queries like:
- *"Show me our carbon footprint totals for Q4 shipments."*
- *"Identify which supplier contributed the most to our embedded carbon."*
- *"Calculate potential CBAM border duties if the carbon price climbs to 90 EUR/ton."*
